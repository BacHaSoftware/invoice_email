## About The Project

<div align="left">
  <a href="https://github.com/BacHaSoftware/invoice_email">

#### Key Features:

üåü <code>Send Invoice By Email</code>: By default, Odoo only allows sending invoices via email to many partners that each partner receives one email. This module provides a solution to help you send an email to many partners, with one recipient and cc to the rest.
